\# Sistema de Inventário com IA



Sistema desenvolvido em Python com interface gráfica (CustomTkinter), base de dados SQLite e previsão inteligente de stock.



\## Funcionalidades

\- Gestão de produtos

\- Dashboard com estatísticas

\- Alertas inteligentes de stock

\- Gráficos automáticos



\## Tecnologias

\- Python

\- CustomTkinter

\- SQLite

\- Matplotlib

\- Pandas

\- Scikit-learn



\## Como executar

pip install -r requirements.txt

python main.py
